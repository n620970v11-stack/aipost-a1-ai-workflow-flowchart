# 繪圖備註
- 請使用 `open_drawio_xml`
- 直接生成 draw.io XML（mxGraphModel）
- 不要使用 `open_drawio_mermaid`
- 不要使用 Mermaid 作為中間格式
- 外部串接系統需明確標示
- 需避免元件重疊
- 方框之間應保留足夠間距
- 流程線應避免穿越或覆蓋流程方框
- 若流程步驟中有呼叫外部系統（含 MCP Server），請在「流程步驟」欄位中補充標注資料傳送方向，例如：「系統調用 MCP Server 查詢設備清單（回傳：設備清單）」，以利繪圖時標示箭頭標籤
